### 代码以及上位机程序

连线方式是：
Arduino Nano <-> 传感器 (DS3231 带纽扣电池那个)
GND <--> GND
3.3 <-> VCC
A4 <-> SDA
A5 <-> SCL

湿度传感器 (蓝色的，不接也可以,代码不用改)
面朝有网格的那一面，从左到右依次是
VCC DATA NC GND
其中VCC，GND接法同上，DATA接D2，NC空着
